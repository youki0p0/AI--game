============================================================
 神楽マキナ OBS オーバーレイ（走る／ジャンプ／飛び続ける）
============================================================

透過背景で、神楽マキナが画面下を走り回り、ジャンプして羽ばたき、
飛び続けるアニメーション素材です。OBSのブラウザソースに入れるだけで動きます。
（kagura_overlay.html 1ファイルだけで完結。画像は埋め込み済み・ネット接続不要）

------------------------------------------------------------
■ OBS での使い方
------------------------------------------------------------
1. OBS を開く
2. 「ソース」→ ＋ →「ブラウザ」を追加（名前は任意）
3. プロパティで：
   - 「ローカルファイル」にチェック
   - 「参照」→ この kagura_overlay.html を選択
   - 幅 / 高さ：配信画面に合わせる（例 1920 × 1080）
     ※キャラは画面下を左右に動くので、画面全体サイズが推奨
   - 「OK」
4. これで画面下を走り回り、たまにジャンプ→飛び続けます。

★背景は透過なので、ゲーム画面や配信レイアウトの上に重ねられます。
★ブラウザでこのHTMLを直接開いても動作確認できます。

------------------------------------------------------------
■ 動きの調整（任意）
------------------------------------------------------------
kagura_overlay.html をメモ帳などで開き、上の方にある
「const CFG = { ... }」の数値を変えると挙動を調整できます。

  speed       走る速さ
  flySpeed    飛ぶ速さ
  flyTime     飛び続ける秒数（大きくするほど長く飛ぶ。例: 20）
  cruiseH     飛ぶ高さ(px)
  flyBobPx    飛行中のふわふわ上下の大きさ
  jumpH       ジャンプの高さ
  runTime     1回の走る時間（秒）
  stepFps     走りのコマ切り替え速度

例）ずっと飛ばせたい → flyTime を 60 などの大きい値にする
